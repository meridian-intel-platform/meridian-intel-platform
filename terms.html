2. Platform description

Meridian Intel is a Retrieval-Augmented Generation (RAG) platform that allows users to upload documents and query them using natural language. The Platform provides:

  • Document ingestion, chunking, and vector storage
  • Hybrid semantic and keyword retrieval
  • AI-generated responses grounded in uploaded document content
  • PII scrubbing and data protection features
  • Evaluation scoring (faithfulness, answer relevance, context recall)
  • Document and query history (tier-dependent)

The Platform operates across five tiers: Ephemeral (Free), Segment, Lattice, Vector, and Enclave (Enterprise). Features and limits vary by tier as described on the pricing page.
